React 中的 路由

SPA应用

  • SPA应用是指单页面应用，只有一个页面，只是不同的组件构成不同的页面效果。

  • 点击页面的某处按钮或链接，不会发生页面跳转，只会局部刷新。

  • 数据通过 ajax请求获取，在后端展现出来。

前端路由

  • 原理：点击路由链接，引发导航栏地址的 path发生变化 。React监测到 导航栏路径发生变化，根据path 选择对应的组件挂载到页面上。

  • 一个路由 对应一个映射关系： key:value key是 导航栏上的/path ， value是一个组件

React是如何检测到导航栏的 path发生变化的？

  • 通过浏览器的历史记录。
